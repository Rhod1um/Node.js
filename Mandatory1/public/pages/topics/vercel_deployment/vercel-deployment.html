Deployment on Vercel

In the bottom of the app's .js file export it in the commonJS format.

            module.exports = app

create a file vercel.json in the root of the project folder.

Copy/paste this code and specify the version of vercel and your app's main .js file, here we use app.js, in the json.

            
                {
                    "version": 2,
                    "builds": [
                      {
                        "src": "app.js",
                        "use": "@now/node"
                      }
                    ],
                    "routes": [
                      {
                        "src": "/(.*)",
                        "dest": "app.js"
                      }
                    ]
                  }

Push your app to your repository.

Log in to vercel using your Github, Gitlab, or Gibucket account.

Upload your project, specify its root folder, and deploy.

If you push your changes but don't see them updated it can because your browser caches the old version.
