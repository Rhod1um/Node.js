Create Node Server with Express

We are assuming you already have node.js installed.

Create a folder which is going to be the project folder.

Create a file called app.js.

cd into the folder and initialize npm in it.

            npm init

Answer the prompted questions.

It creates a package.json file which is the entrypoint for a node.js project.

In the package.json file add ”type”: ”module” before "main": "app.js".

            
              ”type”: ”module”

Install express in your project.

          
            npm i express

Now you should see a node_modules folder in your project folder.

You have now created a Node.js server with express.
